# 3.	TARGET HARDWARE REQUIREMENTS

Athena-IV SBC with the following specifications: 

* E3845 1.91GHz CPU with 4GB RAM 
* 4x USB 2.0 & 1x USB 3.0 ports 
* 4x multi-protocol serial RS-232/422/485 
* 2x Gigabit Ethernet ports 
* 1x SATA port + M.2 2230 SATA socket 
* PC/104-Plus ISA 
* LVDS LCD display 
* VGA CRT 
* HD audio 
* The command line demo programs for the SBC on the image
