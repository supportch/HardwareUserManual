## 8.2 Power IN Connector

The power IN connector is an 2x4 pin custom version of Samtec IPL1 series. The input voltage range is +10V to +28V. Typical input voltage is 24V.

The pinout is as given below:

| SIGNAL NAME | PIN NO | PIN NO | SIGNAL NAME |
| :--- | :--- | :--- | :--- |
| GND | 1 | 5 | GND |
| VIN | 2 | 6 | VIN |
| GND | 3 | 7 | GND |
| VIN | 4 | 8 | VIN |

Connector Part number : **ASP-194529-01**

Mating cable part no:  **DSC No**.**6980512**

## 8.3 External Battery Connector

External battery connector is a 2 pin right angle connector to support real-time clock.

Battery V+ = 3V typ

The pinout is as given below:

| PIN NO | SIGNAL NAME |
| :--- | :--- |
| 1 | Battery V+ |
| 2 | Ground |

Connector part no. : **Molex Mini-SPOX 22057025**

Mating cable Part no. : **DSC No. 6980524**

## 8.18 Utility Connector

Gemini has a utility connector which provides access to the power button, reset button, I2C signals, LPC signals. LPC signals are provided for optional ISA expansion for customers.

{% hint style="info" %}
GbE LEDs are provided optionally instead of LPC signals. The LED signals can be selected as BoM option.
